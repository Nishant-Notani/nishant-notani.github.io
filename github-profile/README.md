### Featured work

**🔌 UGVCL Real-Time Smart-Meter Lakehouse** — *Tech Lead, Government, 3-month onsite*
Built a fully open-source data lakehouse for a government electricity company with **1M+ smart-meter customers** across 4 zones. Kafka → Delta on S3 → Trino → live reporting site, refreshing every 5–15 min.
**Stack:** Apache Kafka · Delta Lake · AWS S3 · Trino · Apache Airflow · Docker
**Impact:** ingested **1B+ raw records/day**, filtered to **50M+/day** stored in raw layer · **20+ dashboards (80+ reports)** delivered · client expanded scope from 60 → 200 dashboards based on delivery quality.

**🏋️ Enterprise Gym Data Platform** — *Architect, 10,000+ locations, 16 SQL Server sources*
End-to-end CDC platform: Fivetran replicating 16 SQL Servers into S3 (Iceberg), Databricks Unity Catalog as the metadata layer, curated ETL in Databricks, analytical layer via Snowflake Stored Procedures + Tasks (8-hour refresh). Built a QuickSight Embedded POC (Streamlit + FastAPI) in 2 weeks.
**Stack:** Fivetran · Apache Iceberg · Snowflake · Databricks Unity Catalog · QuickSight Embedded · FastAPI · Streamlit

**🟦 Microsoft Fabric Enterprise Warehouse** — *Lead, 100+ locations, team of 5*
Cloud-native data warehouse on Microsoft Fabric for a gym & fitness client. Three-layer Lakehouse + Warehouse architecture, ingestion from 28 data sources via Copy Activity, transformation notebooks powering self-serve Power BI through views.
**Stack:** Microsoft Fabric (Lakehouse, Data Warehouse, Notebooks) · Power BI

**🤖 Mlsense — ML Platform & Data Integration** — *Data Engineer*
Integrated PostgreSQL, Azure SQL Server, and MongoDB into the Mlsense ML platform. Built imputation, encoding, outlier detection. Converted feature engineering from Pandas to Spark SQL — **+40% pipeline performance**.
**Stack:** Python · Django · PySpark · PostgreSQL · MongoDB
